VLOOKUP & HLOOKUP — Exact Match by ID
LookupMedium

Enter the computer brands table. Use VLOOKUP to find Product by ID and HLOOKUP to find Brand by ID — both with exact match.

📋 Main Data Table:
IDBrandProduct
102HPPrinters
103DellMouse
104DellKeyboards
105LogitechCameras
109LogitechDesktops
🔍 VLOOKUP — Find Product by ID:
IDFormulaProduct
104=VLOOKUP(104,A:C,3,0)Keyboards
109=VLOOKUP(109,A:C,3,0)Desktops
🔍 HLOOKUP — Find Brand by ID:
IDFormulaBrand
104=HLOOKUP(104,A1:C6,2,0)Dell
💡 The last argument 0 (or FALSE) means exact match. Always use 0 when IDs are not sorted!
